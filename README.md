# Maquetación UI estilo WhatsApp — Entrega

## Descripción
Este proyecto consiste en la maquetación de una interfaz gráfica inspirada en WhatsApp Web.  
Se reprodujo la estructura general de la aplicación, incluyendo el sidebar con la lista de chats y el panel principal de conversación.

El desarrollo se realizó utilizando únicamente **HTML y CSS**, sin el uso de JavaScript, enfocándose en la organización del layout y los estilos visuales.

---

## Estructura del proyecto

/styles/main.css
/index.html  
/README.md

---

## Cómo visualizarlo
Abrir el archivo `index.html` con live server.

---

## Decisiones de diseño y consideraciones
- Se utilizaron **colores similares a la interfaz original de WhatsApp**, adaptados a una versión simplificada.
- La tipografía fue elegida para mantener una apariencia clara y legible.
- Se empleó **Flexbox** para organizar el layout principal y alinear los elementos de forma responsiva.
- Los estilos fueron organizados en secciones dentro del archivo CSS para facilitar su lectura y mantenimiento.
